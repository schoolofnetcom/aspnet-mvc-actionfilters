# ASP.NET MVC - Action Filters

Projeto destinado aos desenvolvedores ASP.NET MVC, com o objetivo de apresentar os Action Filters, que são lógicas de filtragem que serão inseridas e programadas para executarem antes ou depois da chamada de um Action Method.

Você insere no seu projeto para incluir um comportamento aos métodos da sua Controller, de forma a compor o comportamento padrão que seu Action Method já possui no escopo explicito.
